# AI 人物真实感·面部局部特写关键词库

> 来源：抖音合集「AI视觉真实感系列」（作者：啊布 / AI短剧实战派，目前 3 集）。第 1 集为图文（眼睛特写，文案 + 关键词信息图），第 2 集为视频（微表情，核心框架以画面文字卡呈现、音轨无有效语音），第 3 集为视频（皮肤 / 毛孔，音轨转写）。由文案、画面文字与图片信息图蒸馏整理为可直接套用的关键词配方。
>
> **定位**：专攻「AI 人物一眼假 → 一眼真」的**面部局部真实感关键词库**。与 [prompt-craft-and-realism.md](prompt-craft-and-realism.md)（讲去 AI 感的底层机制与减法审美）互补：那篇讲「为什么假、机制怎么控」，这篇给**按面部部位分组、可照抄的真实感触发词**，以及人物情绪「微表情」的过程化写法。适用于 Nano Banana Pro / GPT-image 等图像模型的人物资产图，也适用于 Seedance 2.0 人物镜头。所有写法均为正向、可量化描述，符合 SKILL.md 铁律 5（审核规避）。
>
> 一句话内核：**AI 人物「一眼假」的根因不在五官，而在「太干净、太完美」——真实感来自可控的不完美。**

---

## 一、为什么 AI 人物「一眼假」（先认清错误特征）

| 部位 | AI 一眼假的典型错误 | 真人其实都有的细节 |
|------|---------------------|--------------------|
| 眼睛 | 眼睛太亮、眼白太干净、睫毛像刷子 | 微红血丝、眼周纹理、瞳孔高光、不完美睫毛、自然毛流眉毛 |
| 皮肤 | 过度磨皮、玻璃肌、假白、零毛孔、没有皮肤层次、太光滑太均匀「像建模 / 假人」 | 毛孔大小不一、绒毛分布不规则、鼻翼会出油、脸颊会泛红、高光不会均匀反射 |

> 原则：真人皮肤看起来真实**不是因为完美，恰恰相反**——是因为有这些不规则细节。去 AI 感 = 主动写「可控的不完美」（呼应 prompt-craft-and-realism.md「误解机制」：要去掉某效果直接写「无 X」，要加真实感直接写出具体瑕疵）。

## 二、面部局部特写关键词库（按部位分组 · 照抄即用）

> 系列原作把人物真实感拆成「100 例 · 7 大组别」，逐例给「关键词 + 提示」。以下整理已公开的 3 组（眼睛 / 嘴唇 / 鼻子+脸颊），每条都是一句可直接拼进 prompt 的描述。

### 2.1 眼睛特写

| 关键词 | 可照抄提示 |
|--------|-----------|
| 瞳孔高光 / 红血丝 / 睫毛 | 真实眼球，高光清晰，轻微红血丝，眼毛自然稀疏 |
| 眼周纹理 / 眼下细纹 | 眼周皮肤纹理，眼下细纹自然，生活化光影 |
| 眉毛毛流 / 眼神光 | 眉毛毛流清晰，眼神有光，眼皮微肿，像手机拍摄 |

### 2.2 嘴唇特写

| 关键词 | 可照抄提示 |
|--------|-----------|
| 唇纹 / 湿润高光 | 唇部自然湿润，唇纹清晰，带湿润高光 |
| 轻微干纹 / 嘴角阴影 | 唇部轻微干纹，嘴角有自然阴影，不完美更真实 |
| 牙齿微露 / 真实感 | 嘴唇微张，牙齿微露，唇色自然，不要过度饱和 |

### 2.3 鼻子 + 脸颊特写

| 关键词 | 可照抄提示 |
|--------|-----------|
| 鼻翼阴影 / 毛孔 | 鼻翼有自然阴影，毛孔可见，皮肤有油光 |
| 脸颊泛红 / 绒毛 | 脸颊轻微泛红，皮肤细小绒毛，真实肤质 |
| 肤色不均 / 局部瑕疵 | 肤色不均匀，局部有小瑕疵或痘印，更有真实感 |

> 系列后续组别（同法继续拆解，写人物特写时可预留）：半张脸、发丝遮挡、侧脸、回头瞬间。
>
> **小技巧（通用增强）**：在以上关键词后追加「细腻光影」「高清细节」「电影质感」等词，可大幅提升画面质量和真实感。

## 三、皮肤 / 毛孔真实感关键词清单（去「塑料感」核心）

让 AI 生成真实皮肤，在 prompt 里加入这一组关键词（按需取用、不必全堆）：

- 真实皮肤纹理、可见毛孔、毛孔清晰可见
- 细小绒毛、脸颊轻微泛红、肤色自然不均匀
- 鼻翼有自然油光反射、自然油光、局部泛红
- 自然光拍摄、iPhone 原生相机效果 / 手机原相机质感、未经修图、无滤镜、生活化抓拍、8K 高清

> 目标态：毛孔分布、脸颊绒毛、局部泛红、自然油光、肤色不均，连高光反射都接近真人——皮肤层次到位，人物就从「假人 / 建模感」变「抓拍真人感」。

## 四、微表情：不要写情绪，要写「情绪变化过程」

人物「一眼真 vs 一眼假」在动态镜头上的分水岭：**大多数人只写情绪（结果），不写情绪变化过程**。问题往往不是模型不行，而是 prompt 只写结果、没写过程。

**微表情提示词框架（8 步）**：

`锁定人物 → 锚定初始表情 → 拆解肌肉变化 → 设计情绪递进 → 添加自然动作 → 控制节奏 → 强化质感 → 反向限制 Bug`

- **锁定人物 / 锚定初始表情**：先定「她现在是什么表情」，给一个明确起点（如：平静、勉强微笑）。
- **拆解肌肉变化 + 设计情绪递进**：把情绪拆成可见的肌肉动作序列，写「过程」而非「难过」二字——例如 `先强忍微笑 → 眼神短暂避开 → 嘴角慢慢放下 → 呼吸变浅 → 最后低头紧握手指`（与 cinematic-techniques / seedance2-prompt-cookbook 的「情绪递进拆可见动作」一致）。
- **添加自然动作 + 控制节奏**：配合细微动作与停顿，让递进有节奏，不要一步到位。
- **强化质感**：叠加第二、三章的面部 / 皮肤真实感关键词，保证特写经得起放大。
- **反向限制 Bug**：用正向「禁指向」收尾，减少常见崩坏（如 `不要夸张哭喊、表情克制、眼神变化明显`），呼应 seedance2-prompt-cookbook 8.2「禁指向≤3-5 个」。

> 这正是「高级感来自克制」：打动人的不是大哭，而是「想哭却忍住」的过程被逐帧写了出来。

## 速查总表

- **要眼睛真** → 瞳孔高光 + 微红血丝 + 不完美/稀疏睫毛 + 眼周纹理 + 眉毛毛流 + 眼神光
- **要嘴唇真** → 唇纹 + 湿润或轻微干纹 + 嘴角自然阴影 + 牙齿微露 + 唇色不过饱和
- **要鼻颊真** → 鼻翼阴影 + 可见毛孔 + 油光 + 脸颊泛红绒毛 + 肤色不均/局部瑕疵
- **要皮肤不塑料** → 真实皮肤纹理 + 可见毛孔 + 细小绒毛 + 自然油光反射 + 肤色不均 + 手机原相机/未修图/无滤镜 + 8K
- **要情绪真（动态）** → 写情绪变化过程，不写情绪词；按 8 步框架拆肌肉变化 + 递进 + 克制
- **通用增强** → 关键词后加「细腻光影 / 高清细节 / 电影质感」
