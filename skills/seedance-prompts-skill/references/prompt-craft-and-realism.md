# 提示词工程底层机制与去 AI 感技巧库（邪修篇）

> 来源：抖音合集「刺猬星球superi · ai创作者的乌托邦」（共 37 集），由视频音轨转写后蒸馏整理为可直接套用的提示词规则。
>
> **核心理念**：AI 图 / 视频"一眼假"，根因不是模型弱、也不是提示词不够长，而是你在**"许愿"而非用视觉语言下指令**——把感觉、情绪、风格标签丢给 AI，让它替你猜。本篇讲的是**提示词的底层机制（"邪修"控制法）**与**破除 AI 感、做出真实感**的工程化方法。与 [cinematic-techniques.md](cinematic-techniques.md)（视听语言 / 镜头叙事）互补：那篇讲"怎么用镜头讲故事"，这篇讲"怎么精准控制 AI、让画面可控且不假"。
>
> 适用范围：既适用于资产图（Nano Banana Pro / GPT-Image 等）的提示词，也适用于 Seedance 2.0 视频分镜提示词。所有写法均为正向、可量化描述，符合 SKILL.md 铁律 5（审核规避）。

---

## 一、提示词 = 视觉语言，不是许愿

- **别写"许愿句"**："一个很美的画面、电影感、高级感"——AI 只能猜。提示词是一门视觉语言，你不学会用视觉说话，AI 就只能瞎编。
- **五板块结构（保持结构完整才有可控性）**：`情绪 + 主体 + 光线 + 镜头 + 细节`。
- **具体永远比简短有力**：不写"光线好"，写"85mm 镜头、平视构图、轻微压缩感"。
- **专业词汇带来专业结果**：要质感 →"哑光陶瓷质感、釉面折射质地"；要构图干净 →"三分法、负空间"。给 AI 的视觉信息越多，它瞎编越少。

## 二、AI 底层机制"邪修"控制法（核心）

这一组是本系列最独特的部分——利用模型的行为机制反向控制画面。

### 1. 误解机制（你说 A，AI 理解成 B）
- **缺省词被误解成默认效果**：写"干净"它加纹理、写"自然"它加锐化。→ **直接告诉 AI 不要什么**：`无纹理、无锐化`，越明确越听话。
- **抽象词被误解成堆装饰**：写"梦幻"它加烟雾、写"高级"它塞光斑。→ **把抽象词拆成可量化视觉动作**：`梦幻 = 低饱和 + 软光源 + 长焦压缩`；`高级 = 强光线对比 + 部分留白`；`质感 = 微弱侧光 + 高阴影对比`。
- **不确定描述触发自动补齐**：写"自然感/真实感"它乱补背景、环境、细节。→ 越精准限制，AI 越不敢乱画。

### 2. 词序等级（语序就是优先级）
- **主体词放最前**：AI 优先解析靠前的词，主体词前置 → 重点不跑偏。
- **光影词放中段**：放末尾 AI 会"不分主体随便补光"；放中段才能让光精准作用在主体上。
- **抽象词放末尾**：放前面会被当全局指令破坏主体与光影；放末尾只做整体微调（最好再拆成可量化操作）。

### 3. 扰动词（无意义干扰信号，分流注意力）
- 一个不描述画面、没有具体含义的词，会被 AI 优先处理、改变注意力分配。用途：
  - 防**主体过度优化**（死磕细节导致发糊噪点）；
  - 防**抽象词被夸张化**（在抽象词前加扰动词分流）；
  - 防**自动补齐覆盖主体**（先处理扰动词再画主体，主体更清晰）。

### 4. 鲁棒性破坏（Robustness Break：可控失控 → 更松弛有张力）
- **构图失控**：把"精确构图"换成"趋势词"（主体位置、留白区域、视觉重心），让构图小幅飘移更自然。
- **光线失控**：用"多光源 / 混合光"给 AI 不稳定输入，制造真实的折射与张力。
- **风格失控**：风格词会让 AI 套数据库里最典型的模板（越来越"AI"）；加入打断词让它重新思考画面，风格更特别。

### 5. 伪透视词（AI 不懂透视，只是模仿透视）
- **方向词控延伸**：用"对角延伸感、向外拉伸"让 AI 误以为有被拉长的趋势线。
- **密度词控远近**：用"近处密、远处疏"让 AI 自动分层出纵深，不必堆环境词。
- **尺度词控比例**：用夸张的相对尺寸落差，得到强透视冲击。

### 6. 时间词（时间补偿机制 → 情绪与动态）
- **过去式触发情绪残留**："刚刚 / 曾经"让 AI 模拟前几秒、延续光影与情绪。
- **现在式制造动态**："正在 / 此刻"让场景被当成进行中的事件，主体背景有运动趋势与呼吸感。
- **将来时触发张力**：未来的不确定性让 AI 推演下一瞬，强化紧张与压迫。

### 7. 特征塌陷（深层渲染会自动压缩某些特征 → 控制情绪浓度）
- **压缩非主体特征**：显式弱化次要元素 / 降低细节密度，情绪更集中。
- **特征衰减制造张力**：`轻微塌陷 + 动态偏移 / 边缘柔化` → 微妙模糊与飘移，画面更高级。
- **强化局部对比**：在关键区域强调光影落点 / 色彩重点 / 动作焦点，主体与情绪鲜明。

### 8. 反着写提示词（镜头前置，包裹动作）
- AI 是**按顺序执行**提示词的，不是先理解全局。正常"先动作→再细节→最后镜头"会让画面像拼出来的。
- **反过来写**：先写镜头（`低角度镜头缓慢推进 / 手持轻微晃动`），再写人物动作 → 等于告诉 AI"这些动作在同一个镜头里完成"，更连贯真实。进阶："用镜头包裹动作"，并把自己想象成一台摄像机给出参照方向（解决 AI 没有方向感的问题）。

## 三、光线控制（AI 图最容易露馅的地方）

- **基础三步**：① 光的**方向 / 来源 / 阴影布局**（别只写"摄影灯光"，太笼统）；② 光**比**（明确哪里亮哪里暗，靠明暗对比撑真实感）；③ **色温**（偏冷=清晨雾、偏暖=傍晚风；冷暖交错有情绪）。
- **进阶三步**：① **体积光**（光穿过什么介质→变成怎样的光束）；② **反光与反射色**（反射来源点 / 反射到哪 / 反射光颜色，拉满材质感）；③ **补光层次**（主光在哪、补光保留多少，让光有主次）。

---

## 四、JSON 结构化生图（JSON 负责"可控"，不负责"审美"）

- 误区：以为写成 JSON 画面就会变好看。**JSON 只让输出更可控、更可复用，不提升审美。**
- **结构框架**：第一行必须是 `subject`（整张图的锚点），再 `environment`，最后 `style`；一上来写风格容易带着画面乱跑。
- **三步法**：① 先研究目标美学（拆解一部参考电影的相机 / 镜头 / 画幅 / 色彩，如柯达胶片、富士冷调）；② 写 JSON 锁定构图 / 灯光 / 渲染质量，加约束（`无数码锐化、不生成文字`），可让 LLM 把同一套适配到不同场景；③ 让画面动起来（先环绕再推进再剪辑）。
- **何时用 JSON**：偏**工作流**的工具里 JSON 是生产工具（把一次成功复制成稳定批量）；快速试风格时自由文本更高效。先要有**好的创意参考**，JSON 才有意义。

## 五、反推与风格提取（别把"结果"当"原因"）

- **反推画面提示词**：① **锁定不变项**（先想清为什么被这张图吸引——色彩 / 构图 / 情绪，强调要保留的效果）；② **生成结构骨架**（风格词只是装饰，真正贯穿的是构图、光照方向、主体与背景关系）；③ **验证纠错**（让 AI 用反推词先生成一张，再对比原图自我分析，找出失效与起作用的条件）。
- **提取风格**：① **看"没什么"而非"有什么"**（问 AI"生成时哪些是不可变条件：构图、视角、空间关系"）；② **看共性而非单张**（多找几张同风格不同内容的图让 AI 找共性，反复被保留的选择才是风格）；③ **复用结构搭体系**（拆成"空间与构图 / 光线与氛围 / 色彩与质感 / 限制与规则"四板块，做成情绪板长期执行）。风格 = 一套被反复执行的选择逻辑，不是一句提示词。
- **反推视频提示词**：截一张图让它动 ≠ 复现。要 ① 反推**生成过程**而非画面内容；② 用**相机运动结构**反推（焦距如何伸缩、坐标如何偏移，要参数化而非感性意境）；③ **具象化指令**（把感性长句翻译成"动作 + 数值"的执行格式，剔除废词）。

## 六、去 AI 感 / 真实感（"AI 最大的问题不是不真实，而是太干净"）

- **去摆拍感（视频）**：摆拍感来自镜头与画面关系不真实，与提示词无关。三因：① 取景起点完成度过高（让画面从不完整视角开始：前景遮挡 / 构图偏移，再用运镜去"找"画面）；② 运动轨迹过度平滑（引入减速 / 停顿等人为操作痕迹）；③ 主体动作与镜头完全同步（让**人物动作早于镜头反应**，制造"被进入的瞬间"）。
- **前景遮挡去 AI 味**：① **物理遮挡**（树叶 / 柱子 / 人物背影 + 低角度 + 前景虚化；注意 AI 遵循顺序机制，不写"虚化"它会把前景画清楚而失去景深）；② **氛围遮挡**（让空气里有东西：雪花 / 雾气 / 颗粒感，给画面厚度）。
- **环境真实感**：① 拆**空间层次**（明确前景 / 中景 / 远景，别用一个标签概括）；② 写**逻辑因果**（"因为背景灯照亮墙面、因为绿植遮挡所以有阴影"，避免拼贴感）；③ **光线与天气**（时间 / 光源类型 / 光强方向；天气是环境状态的一部分）。
- **电影感 = 摄影约束先于美学**：① 先写**镜头类型 / 拍摄距离 / 光线结构**，再写风格（90% 的人上来就写氛围，AI 只停在"图像渲染"而非"真实摄影"）；② 把情绪翻译成**摄影语言**（不写"孤独"，写"人物置于画面边缘、大量负空间、被环境压"）；③ 用**生成顺序**制造电影感（第一轮只生成构图+光线 → 加人物动作 → 最后加风格）。
- **镜头语言匹配情绪**：① **景别控心理距离**（远景=情绪去参与化，适合孤独 / 渺小；中景=真实社交距离、蓄势；特写=放大情绪，但需远 / 中景铺垫）；② **焦段控空间压迫**（广角夸张近大远小→侵略 / 压迫；中长焦统一空间→高级电影感）；③ **统一空间逻辑**（景别 / 焦段 / 构图服务同一目标，避免视觉信息打架）。
- **减法审美**：审美降级时代，信息越多越廉价。剪掉**复杂背景**（明确主体与背景分层）、**复杂光影**（只留关键光源 + 明确光影因果）、**杂乱色彩**（限定主色 + 辅色，删干扰色）。
- **剪词（让画面显贵）**：① 剪**情绪词**（"高级感 / 电影感"对 AI 仍非有效控制信息，模型变聪明 ≠ 词变精准，本质还是抽卡）；② 剪**重复词**（同义反复只会分散权重，每个信息只出现一次）；③ 剪**人类句式**（别写成讲故事的整句，拆成元素：谁在动 / 怎么动 / 镜头固定还是跟随）。

## 七、调色（"在 AI 时代，感觉不可靠，可靠的是数据"）

- **三步法**：① 把审美**叫为数据**（别给"某某电影风格"标签，拆成色相分布 / 明暗比例 / 色温倾向）；② 用 **JSON 代码控色**（把色彩数据整理成 JSON 调色规则再执行，避免误差放大）；③ **验证**（把调色图与参考图一起交给 AI 对比色彩特征，一致才算对，不一致让它指出该调哪里）。
- **Hex 自动调色法**：上传几张参考图，让模型扫描提取 Hex 色值生成专属调色板，生成时选用该色板 → 任何提示词都自动带相同标志性色彩。本质是**把颜色控制从"后期"提前到"生成之前"**。

## 八、动作（崩坏与打击感）

- **避免动作崩坏**：AI 眼里动词是"目标状态"不是"进行态"。① **减动词、增方式词**（保留一个核心动作，用"犹豫的 / 持续的 / 重心转移"限定节奏，避免指令冲突）；② **单一主动作锁身体结构**（同一时刻只给一个主动作，其余从属：走路回头看的主动作是 walking，回头是朝向变化）；③ **顺序翻译成状态约束**（别用"然后 / 接着 / 同时"，深层模型不可靠地执行时间轴；改描述"动作发生时角色处于什么状态"）。
- **打击感**：本质是视觉欺骗 + 有结构的动作。① 写**动作前摇**（重心变化 / 短暂停顿，AI 才知道接下来要爆发）；② **少形容词、多画面解释**（"刀光剑影"被 AI 理解成火花特效；直接解释"武器高速交错、攻击方向"）；③ **镜头与环境反馈**（命中瞬间镜头震动 / 运动模糊 / 环境元素位移，否则 AI 默认镜头理性稳定）。

---

## 九、一致性（人物与声音）

- **人物一致性（视频）**：参考图不是"参考人物"，只是风格与结构输入。① **人景拆分**（先生成人物多角度三视图成为稳定可复用对象，别和场景一起生成）；② **先动作、再进场景**（先让人物完成指定动作再与场景合成，避免叠加场景时重推人物结构）；③ **拆镜头降时间维度**（连续帧每多一帧多一次偏移，尽量一个动作一段视频）。
- **声音一致性**：AI 视频最易露馅的是声音。① **建角色声音参考库**（把满意片段的音频抽出存成独立文件，每次生成上传作参考）；② **独立配音模型设计音色**（把角色图 + 性格交给配音模型设计并锁定长期使用）；③ **建情绪声音库**（同一角色生成"正常 / 愤怒 / 伤心"等情绪样本，按剧情选用，情绪表现也稳定）。

## 十、工作流与导演思维（从"出片"到"创作"）

- **AI 是生产系统，不是输出工具**：90% 的人把 AI 当一次性出图工具。建**可复用工作流**才能高效产出（如时尚品牌流程：可复用虚拟模特 → 参考图锁定姿态 → 模板复用成封面 / 动态视频）。
- **Cinema Studio 思路**：选一台真实存在的电影摄影机（Arri Alexa / RED）+ 镜头焦段，按真实物理逻辑决定虚化 / 光斑 / 反射，而非随机拼凑 → 真实比例、完整电影语言。
- **废片回收（少抽卡）**：① **拆分有效信息**（满意的人物 / 色彩部分单独作参考，别全图重算）；② **局部重塑**（念图生图 / 蒙版只改不满意处，保持整体完整）；③ **当负面样本**（让 AI 提取废片中的错误元素作负面提示词，下次自动避坑）。
- **导演思维**：① **调度优先于分镜**（先确定空间规则——站位、镜头运动，理解空间关系，而非让 AI 替你做场面调度）；② **叙事优先于画面**；③ **补拍思维**（转折不明显补一个反应特写、节奏太平补一个空镜）。模型只是执行者，你才是导演。

## 速查总表

- **画面一眼假** → 别许愿，用五板块 + 具体 + 专业词；剪情绪词 / 重复词 / 人类句式
- **AI 不听话 / 乱补** → 误解机制（说"不要 X"+ 抽象词量化）、词序等级、扰动词
- **太规整 / 太干净** → 鲁棒性破坏、前景遮挡（物理 + 氛围）、减法审美
- **没空间 / 没纵深** → 伪透视词、环境空间层次 + 因果、焦段控压迫
- **没情绪 / 没动态** → 时间词、特征塌陷、氛围感（明暗 / 冷暖 / 虚实取舍）
- **要电影感** → 摄影约束先于美学、镜头语言匹配情绪、Cinema Studio、生成顺序
- **要可控可复用** → JSON 结构化、风格提取四板块、调色数据化 / Hex、可复用工作流
- **动作崩 / 没打击感** → 减动词增方式词、单一主动作、前摇 + 镜头环境反馈
- **人物 / 声音变** → 人景拆分 + 三视图、拆镜头、声音参考库 + 情绪库
- **想真正变强** → 导演思维（调度 / 叙事 / 补拍），把 AI 当生产系统
