# AI 人物真实感·面部局部特写关键词库

> 来源：抖音合集「AI视觉真实感系列」（作者：啊布 / AI短剧实战派，目前 5 集）。第 1 集为图文（眼睛特写，文案 + 关键词信息图），第 2 集为视频（微表情，核心框架以画面文字卡呈现、音轨无有效语音），第 3 集为视频（皮肤 / 毛孔，音轨转写），第 4 集为图文（发丝特写，「AI 发丝真实感 100 例 · 7 大组别」关键词信息图 + 文案），第 5 集为视频（美妆模特嘴唇，音轨转写 + 文案）。由文案、画面文字与图片信息图蒸馏整理为可直接套用的关键词配方。
>
> 另补充一条同主题单集：Jac.key「别再无脑堆砌提示词，先搭结构再打光 + 多角度」（见第五章）。
>
> **定位**：专攻「AI 人物一眼假 → 一眼真」的**面部局部真实感关键词库**。与 [prompt-craft-and-realism.md](prompt-craft-and-realism.md)（讲去 AI 感的底层机制与减法审美）互补：那篇讲「为什么假、机制怎么控」，这篇给**按面部部位分组、可照抄的真实感触发词**，以及人物情绪「微表情」的过程化写法。适用于 Nano Banana Pro / GPT-image 等图像模型的人物资产图，也适用于 Seedance 2.0 人物镜头。所有写法均为正向、可量化描述，符合 SKILL.md 铁律 5（审核规避）。
>
> 一句话内核：**AI 人物「一眼假」的根因不在五官，而在「太干净、太完美」——真实感来自可控的不完美。**

## 一、为什么 AI 人物「一眼假」（先认清错误特征）

| 部位 | AI 一眼假的典型错误 | 真人其实都有的细节 |
|------|---------------------|--------------------|
| 眼睛 | 眼睛太亮、眼白太干净、睫毛像刷子 | 微红血丝、眼周纹理、瞳孔高光、不完美睫毛、自然毛流眉毛 |
| 皮肤 | 过度磨皮、玻璃肌、假白、零毛孔、没有皮肤层次、太光滑太均匀「像建模 / 假人」 | 毛孔大小不一、绒毛分布不规则、鼻翼会出油、脸颊会泛红、高光不会均匀反射 |

> 原则：真人皮肤看起来真实**不是因为完美，恰恰相反**——是因为有这些不规则细节。去 AI 感 = 主动写「可控的不完美」（呼应 prompt-craft-and-realism.md「误解机制」：要去掉某效果直接写「无 X」，要加真实感直接写出具体瑕疵）。

## 二、面部局部特写关键词库（按部位分组 · 照抄即用）

> 系列原作把人物真实感拆成「100 例 · 7 大组别」，逐例给「关键词 + 提示」。以下整理已公开的 3 组（眼睛 / 嘴唇 / 鼻子+脸颊），每条都是一句可直接拼进 prompt 的描述。

### 2.1 眼睛特写

| 关键词 | 可照抄提示 |
|--------|-----------|
| 瞳孔高光 / 红血丝 / 睫毛 | 真实眼球，高光清晰，轻微红血丝，眼毛自然稀疏 |
| 眼周纹理 / 眼下细纹 | 眼周皮肤纹理，眼下细纹自然，生活化光影 |
| 眉毛毛流 / 眼神光 | 眉毛毛流清晰，眼神有光，眼皮微肿，像手机拍摄 |

### 2.2 嘴唇特写

| 关键词 | 可照抄提示 |
|--------|-----------|
| 唇纹 / 湿润高光 | 唇部自然湿润，唇纹清晰，带湿润高光 |
| 轻微干纹 / 嘴角阴影 | 唇部轻微干纹，嘴角有自然阴影，不完美更真实 |
| 牙齿微露 / 真实感 | 嘴唇微张，牙齿微露，唇色自然，不要过度饱和 |

**美妆商品图专用·嘴唇先露馅（ep05）**：做口红 / 唇釉 / 护肤种草图时，嘴唇一假整张商品图就「像 AI 生成」。别只写 `红唇`、`性感`、`高清美女`、`漂亮嘴唇`——真人嘴唇不是一块红色贴片，至少有四个细节：

| 真人嘴唇细节 | AI 常犯错误 | 可照抄提示 |
|--------------|------------|-----------|
| 唇形轻微不对称 | 太对称像剪贴脸 | `唇形轻微不对称、自然唇形` |
| 嘴角一定有阴影 | 没有嘴角阴影，嘴巴像贴上去 | `嘴角有自然阴影、唇与皮肤自然过渡` |
| 下唇轻微高光 | 高光像塑料 / 亮成玻璃 | `下唇轻微高光、不要塑料光` |
| 唇纹有细小纹理与颜色变化 | 唇纹太干净、颜色太均匀 | `自然唇纹、唇色有层次、局部轻微干燥` |

> **关键框架**：要让 AI 理解「**这是美妆商品图，不是网红自拍**」——出片的不是「漂亮嘴唇」，而是有唇纹、颜色层次、局部干燥、自然反光与嘴角细节的真实嘴唇。

### 2.3 鼻子 + 脸颊特写

| 关键词 | 可照抄提示 |
|--------|-----------|
| 鼻翼阴影 / 毛孔 | 鼻翼有自然阴影，毛孔可见，皮肤有油光 |
| 脸颊泛红 / 绒毛 | 脸颊轻微泛红，皮肤细小绒毛，真实肤质 |
| 肤色不均 / 局部瑕疵 | 肤色不均匀，局部有小瑕疵或痘印，更有真实感 |

> 系列后续组别（同法继续拆解，写人物特写时可预留）：半张脸、侧脸、回头瞬间。
>
> **小技巧（通用增强）**：在以上关键词后追加「细腻光影」「高清细节」「电影质感」等词，可大幅提升画面质量和真实感。

### 2.4 发丝特写（AI 人物 / 商品模特真实感的「天花板」· ep04「100 例 · 7 大组别」）

当皮肤已有毛孔、眼睛已有神、嘴唇已真实，**还是一眼像 AI，多半是头发在出卖它**——头发太整齐 / 太顺滑 / 太完美，「像头盔、像建模、像假人」。发丝是 AI 人像、AI 商品模特 / 电商主图 / 种草图 / 服装模特 / 美妆图最容易露馅、也最能拉开真假的部位。**脸决定点击率，发丝决定真实感**——很多人一直在修脸，但真正让 AI 模特图露馅的其实是发丝。

**真实摄影 vs AI 发丝的分水岭**：

| 要素 | AI 常犯错误 | 真人头发其实都有 |
|------|------------|------------------|
| 发际线 | 发际线太整齐、头发像头盔 | 自然发际线、额前碎发、发际线不规则 |
| 发丝 | 每根头发一样顺、发丝粗细一致、过于整齐 | 发丝粗细不均、几根碎发自然散落、边缘飞发 |
| 边缘 | 发丝边缘像钢丝 / 糊成一片、发丝与皮肤分界太干净 | 发尾不规则 / 自然分叉、发丝遮挡部分五官、头发贴脸 |
| 层次 / 光 / 焦 | 没有层次、缺少碎发细节、整体死顺 | 光线穿过发丝形成透光感（逆光透发）、局部虚焦、风吹后的凌乱感 |

**「AI 发丝真实感 100 例 · 7 大组别」关键词配方**（ep04 信息图逐组拆解，每条都是一句可拼进 prompt 的描述）：

| 组别 | 何时用 | 可照抄关键词 + 提示 |
|------|--------|----------------------|
| ① 额前碎发 | 发际线 / 额头一眼假 | `额前碎发、飞发、自然毛流`；`碎发、细软发丝、光影`；`发际线不规则、原生感、有新长出的细软发` |
| ② 飞发与凌乱感 | 死顺、太规整 | `飞发、凌乱感、风吹发丝`（方向不一、自然凌乱）；`阳光穿过发丝形成透光、层次丰富` |
| ③ 发丝遮挡与脸部互动 | 头发像贴图、不互动 | `发丝遮挡部分五官、形成阴影、增强真实感`；`发丝与脸部互动、增加动态感与故事感`；`自然垂落、贴脸发丝` |
| ④ 发丝质感特写 | 近景 / 商品图主图 | `发丝质感清晰可见、每根发丝粗细不一`；`发丝分叉、自然磨损、常见状态`；`真实发丝、质感强` |
| ⑤ 发丝近景 / 细节 | 特写镜头 | `发丝近景、细节清晰、层次分明`；`自然光泽、真实质感、不假滑`；`轻微毛流、细节丰富、真实自然` |
| ⑥ 发丝与光线互动 | 逆光 / 高级感 | `发丝透光、阳光穿透、梦幻感`；`逆光发丝、光影交错、层次丰富`；`发丝高光、自然光影、质感真实` |
| ⑦ 发丝动态 / 形态 + 头皮 / 发际线 | 形态僵硬、根部假 | `发丝飘动、自然凌乱、动态感`；`发丝形态、自然卷曲、真实感`；`发丝根部 / 头皮清晰可见`；`发丝自然分隔、生长方向不刻意`；`发际线自然过渡、真实细节` |

> **小技巧（信息图原话）**：在关键词后追加 `自然光`、`抓拍感`、`原相机质感` 等词，可大幅提升发丝真实感与细节表现。
>
> 同一条铁律：**发丝越完美越像 AI，发丝越不完美反而越像真人**。AI 商品图越来越像真人，拼的已经不是脸，而是这些细节。

## 三、皮肤 / 毛孔真实感关键词清单（去「塑料感」核心）

让 AI 生成真实皮肤，在 prompt 里加入这一组关键词（按需取用、不必全堆）：

- 真实皮肤纹理、可见毛孔、毛孔清晰可见
- 细小绒毛、脸颊轻微泛红、肤色自然不均匀
- 鼻翼有自然油光反射、自然油光、局部泛红
- 自然光拍摄、iPhone 原生相机效果 / 手机原相机质感、未经修图、无滤镜、生活化抓拍、8K 高清

> 目标态：毛孔分布、脸颊绒毛、局部泛红、自然油光、肤色不均，连高光反射都接近真人——皮肤层次到位，人物就从「假人 / 建模感」变「抓拍真人感」。

## 四、微表情：不要写情绪，要写「情绪变化过程」

人物「一眼真 vs 一眼假」在动态镜头上的分水岭：**大多数人只写情绪（结果），不写情绪变化过程**。问题往往不是模型不行，而是 prompt 只写结果、没写过程。

**微表情提示词框架（8 步）**：

`锁定人物 → 锚定初始表情 → 拆解肌肉变化 → 设计情绪递进 → 添加自然动作 → 控制节奏 → 强化质感 → 反向限制 Bug`

- **锁定人物 / 锚定初始表情**：先定「她现在是什么表情」，给一个明确起点（如：平静、勉强微笑）。
- **拆解肌肉变化 + 设计情绪递进**：把情绪拆成可见的肌肉动作序列，写「过程」而非「难过」二字——例如 `先强忍微笑 → 眼神短暂避开 → 嘴角慢慢放下 → 呼吸变浅 → 最后低头紧握手指`（与 cinematic-techniques / seedance2-prompt-cookbook 的「情绪递进拆可见动作」一致）。
- **添加自然动作 + 控制节奏**：配合细微动作与停顿，让递进有节奏，不要一步到位。
- **强化质感**：叠加第二、三章的面部 / 皮肤真实感关键词，保证特写经得起放大。
- **反向限制 Bug**：用正向「禁指向」收尾，减少常见崩坏（如 `不要夸张哭喊、表情克制、眼神变化明显`），呼应 seedance2-prompt-cookbook 8.2「禁指向≤3-5 个」。

> 这正是「高级感来自克制」：打动人的不是大哭，而是「想哭却忍住」的过程被逐帧写了出来。

## 五、别只堆词：先搭提示词结构，再用「打光 + 多角度」把人物做活

很多人以为只要往提示词里加「真实皮肤、电影质感、高清细节」人物就会变真——但做上百张图后会发现：**关键词加得越多，画面反而越假、人物越僵硬**。真正重要的不是堆词，而是**提示词要有结构**，再配合后期的光线与角度，而不是靠堆词反复抽卡。

**标准化工作流（让人物从「贴在画面里」变「站在场景里」）**：

1. **先搭提示词结构**：按「一张人物图该有的基本逻辑」的标准化结构写好（主体 / 动作 / 姿态 / 光线 / 镜头 / 真实感细节分层，而非一股脑堆形容词），生成第一版——通常已经不错，但「还差一点活人感」。
2. **关键一步·打光**（图像工具的打光功能）：选择打光 → 调光的**强度与颜色**（示例：强度约 60%、默认白光）→ 设光源方位（示例：放在人物**左上方、稍微靠前**）→ 重新生成。对比打光前后，人物会明显从「贴在画面里」变成「真的站在场景里」。
3. **多角度微调**：选择多角度 → 调整角度重新生成。价值是**不用再靠大量提示词反复抽卡**，直接得到不同角度的人物表达——全身体态、肢体小动作、面部微表情。

> 内核：**人物真实感不是靠脸有多高清，而是靠光线、姿态和情绪一起成立**。观众能感受到这个人物的情绪，才会觉得他真实、立体、有代入感。所以别只对着皮肤质感关键词堆——先搭提示词结构，再用打光和多角度把人物真正「做活」。（与 prompt-craft-and-realism.md「减法 / 剪词」、cinematic-techniques.md「光影塑造情绪」一致。）

## 速查总表

- **要眼睛真** → 瞳孔高光 + 微红血丝 + 不完美/稀疏睫毛 + 眼周纹理 + 眉毛毛流 + 眼神光
- **要嘴唇真** → 唇纹 + 湿润或轻微干纹 + 嘴角自然阴影 + 牙齿微露 + 唇色不过饱和
- **要美妆商品图嘴唇真** → 唇形轻微不对称 + 嘴角阴影 + 下唇轻微高光（非塑料）+ 自然唇纹颜色层次 + 唇皮过渡；让 AI 理解「这是美妆商品图不是网红自拍」
- **要鼻颊真** → 鼻翼阴影 + 可见毛孔 + 油光 + 脸颊泛红绒毛 + 肤色不均/局部瑕疵
- **要皮肤不塑料** → 真实皮肤纹理 + 可见毛孔 + 细小绒毛 + 自然油光反射 + 肤色不均 + 手机原相机/未修图/无滤镜 + 8K
- **要发丝真（商品模特同理）** → 7 大组别按需取：额前碎发 / 飞发凌乱 / 发丝遮挡脸部 / 发丝质感分叉 / 近景层次 / 逆光透发光影 / 动态形态 + 头皮发际线；收尾加「自然光 / 抓拍感 / 原相机质感」
- **要情绪真（动态）** → 写情绪变化过程，不写情绪词；按 8 步框架拆肌肉变化 + 递进 + 克制
- **要「活人感」而非堆词** → 先搭标准化提示词结构（别堆形容词）→ 打光（强度/颜色/方位）→ 多角度，靠光线+姿态+情绪一起成立
- **通用增强** → 关键词后加「细腻光影 / 高清细节 / 电影质感」
