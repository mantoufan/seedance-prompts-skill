# Seedance 2.0 提示词实战手册（镜头配方 + 万能公式 + 痛点避坑）

> 来源：抖音合集「Seedance2 教学计划！【小白必看】」（作者：张百川AI，共 18 集），由视频音轨转写后蒸馏整理为可直接套用的提示词规则。原始转写见仓库外的字幕归档。
>
> **定位**：这是一本**面向 Seedance 2.0 的「关键词触发」配方手册**——把每一类镜头 / 光影 / 公式拆成「什么时候用 + 必写触发关键词 + 示例」。与 [cinematic-techniques.md](cinematic-techniques.md)（讲「为什么这样拍」的视听语言）、[prompt-craft-and-realism.md](prompt-craft-and-realism.md)（讲「底层机制 / 去 AI 感」）互补：那两篇讲原理与审美，这篇给**可照抄的触发词与防崩配方**。所有写法均为正向、可量化描述，符合 SKILL.md 铁律 5（审核规避）。
>
> 通用前提（贯穿全篇）：① 动作 / 运镜**宁慢勿快**，模型一旦被催快就崩；② 运镜 / 优先级**一次只突出一个**，别堆叠；③ 单段时长**优先 10 秒**（比 15s 稳、比 5s 完整）；④ 用**多模态 @素材**锁一致性，但「一个素材只担一个职责」。

## 一、运镜配方：写出运镜词就能触发专业镜头

Seedance 2.0 对运镜指令识别力很强，**直接写运镜术语**即可触发。核心是把「景别变化路径 + 情绪目标」描清楚。

| 运镜 | 何时用 | 必写触发关键词 |
|------|--------|----------------|
| 缓慢推进 | 放大情绪、聚焦主体 | `从全景缓慢推至面部特写`、`固定机位`、`推轨` |
| 缓慢拉远 | 张力、孤独、揭示环境 | `从特写拉至大远景`、`人物逐渐缩小`（让模型明确景别从小到大的动态过程） |
| 仰拍 | 气势、压迫、冲击 | `从地面向上抬起`、`强化透视变形`，可加`脚部到头部`全身贴地视角 |
| 俯拍 | 宏大场景、交代环境 | `垂直俯瞰`、`16mm 广角镜头`（强化场景透视感） |
| 跟拍（前/后/侧跟） | 临场真实感、代入冒险 | `侧面跟拍`、`手持轻微呼吸感`（增真实、避免剧烈晃动） |
| 低角度跟拍 | 仰拍 + 跟随的压迫前进感 | 必须**明确机位高度**：`机位高度 0.3 米 / 0.5 米 / 1 米`，别定太高 |
| 希区柯克变焦（滑动变焦） | 天旋地转、心理冲击 | `希区柯克变焦`、`背景被拉伸挤压`（物理推轨 + 反向变焦：推镜加广角端 / 拉镜加长焦端，主体大小不变而背景透视剧变） |

### 1.1 经典「闯入式镜头」（斯皮尔伯格镜头表演法）

短视频爆款反差镜头。灵魂句是主体**突然入画**：`突然从镜头底部猛地战栗入画`——这一句是整条视频的灵魂。配合 `@图片1` 上传人脸 / 角色素材到参考区锁定一致性。

> 真人脸注意：部分 AI 视频工具已限制识别真人面部，上传真实头像可能识别失败；此时改用 AI 生成的角色设定图作参考。

## 二、速度感与特殊镜头配方

| 镜头类型 | 何时用 | 必写触发关键词 |
|----------|--------|----------------|
| 延时摄影 | 把长时间变化压缩进几秒 | `延时摄影`、`云层快速移动`、`一小时浓缩至 15 秒`（让模型理解时间压缩逻辑） |
| 高速跟拍 + 动态模糊 | 锁主体高速运动、背景放射虚化 | `1/50 秒慢快门`、`背景高度动态模糊`（让模型抓住运动重心、避免角色崩坏） |
| 航拍鸟瞰 | 史诗级大环境铺垫 | `FPV 无人机航拍` / `大疆视角` + **高度**（`高度 50/100/150 米`）+ **飞行路径**（`贴地拉升至高空`） |
| 快速变焦推拉 | 暴力美学、打破第四面墙、生理恐惧 | `急速推进拉镜`、`变焦速度每秒 4.5 米 / 5 米`（匹配临场感、增质感） |

## 三、电影级快切与动作类型配方

| 类型 | 何时用 | 必写触发关键词 / 做法 |
|------|--------|------------------------|
| 快速甩镜头 + 转场 | 时间线来回切换、消除剪辑割裂感 | `向右快速甩镜头`、`摇镜至模糊`、`从左侧模糊画面甩入清晰`（模拟无缝剪辑） |
| 碎片化快速剪辑 | 浓缩叙事时间、传递紧张快节奏、大信息量涌入 | 单次生成受限 → **多次生成 + 后期剪辑**；提示词里设 `Shot 1 至 Shot 4 快切序列` |
| 超高速微观摄影 | 肉眼看不到的极快物理现象（子弹击穿等） | `高速摄像机`、`微观爆炸`、`碎片凝固在空中` + **具体主体**（篮球 / 足球 / 汽车，必须具体化） |

## 六、万能公式（照抄套场景）

### 6.1 通用「防崩神词」（几乎每条都加一点）

`画面稳定 · 不闪烁 · 不重影 · 不模糊 · 动作流畅自然 · 不卡顿 · 不断裂 · 光影细腻 · 高清画质 · 电影质感`。这些词稍加几个就能让画面直接上一个档次。

### 6.2 三大固定结构

- **人物公式**：`主体 + 动作 + 表情 + 运镜 + 光线 + 风格 + 保底神词`
  - 模板：`近景特写，面部稳定清晰、五官精致不扭曲、人体结构正常，动作缓慢轻柔且连贯，[光线][风格]，画面稳定不闪烁、高清电影质感`
- **场景 / 风景公式**：`场景 + 视角 + 运镜 + 氛围 + 光线 + 风格 + 保底神词`
  - 模板：`第一视角，春节花灯长廊，红灯笼璀璨流光溢彩，缓慢前进沉浸式穿梭，唯美梦幻写实光影，夜景柔和暖色调氛围，画面稳定不闪烁、无重影、高清画质、丝滑流畅`
- **动漫二次元公式**（AI 漫剧 / 短剧尤其适用）：`动漫角色 + 动作 + 场景 + 画风 + 运镜 + 保底神词`
  - 模板：`二次元动漫少女温柔微笑、缓慢转头，唯美樱花场景、微风飘动发丝，日系清新画风柔和光影，画面稳定不闪烁、动作流畅、高清精致、线条干净不崩坏`
  - 关键词：`画风统一 · 线条干净 · 不崩坏 · 不变形 · 动作轻柔缓慢`

### 6.3 三个简短情绪模板（套场景即用）

- **人物情绪类**：`近景，女生 + [情绪]，缓慢动作，柔和光线，面部稳定清晰，画面不闪烁，动作流畅，高清写实`
- **唯美氛围类**：`唯美场景，沉浸式第一视角，缓慢运镜，暖光氛围，画面稳定、丝滑流畅、高清、电影质感`
- **炫酷氛围类**：`炫酷光影，动感流动，镜头缓慢推进，色彩浓郁，画面稳定不闪烁，高清质感`

### 6.4 三条铁则（公式之外的稳定性原则）

1. **动作写「慢」**：`轻轻地 / 慢慢地 / 轻微地`；模型被催快就乱。
2. **运镜只写一个，不重复不迭代**：`固定镜头 / 缓慢推进 / 平稳跟拍` 选其一；堆叠会让模型理解不了。
3. **时长优先 10 秒**：先别冲 15s 也别只做 5s，10s 最稳也最好看。

## 七、动作描述原则（提升「打的质感」）

Seedance 2.0 对动作流畅度要求极高，精准动作描述直接提质感。

- **核心**：别只写「跳舞 / 走路」，要写**连续性 + 自然 + 慢动作细节**。
- **好用修饰词**：缓慢、轻柔、连贯、自然、流畅。**要避免**：夸张、高速、复杂多人互动、大幅度扭曲（模型理解不到位）。
- **好用动词组合**：人物 `缓慢转身 / 轻轻抬手 / 脚步轻移 / 微微低头 / 随风摆动`；镜头 `镜头缓缓推进 / 轻微环绕 / 平缓跟拍`。
- **景别 + 运镜 + 稳定** 三件套：`中景，缓慢推进，稳定跟拍，画面丝滑不卡顿` → 电影级质感质的飞跃。

## 八、痛点避坑（短剧 / 漫剧实战 · 后台高频问题）

### 8.1 音画 / 口型 / 情绪对不上 → 先用音频定节奏，画面服从声音

Seedance 2.0 支持文本 / 图片 / 视频 / 音频四模态**联合生成**，声音是漫剧的「节奏骨架」。先生成画面再配音 → 口型情绪别扭；正确顺序是**先确定音频**（旁白 / 内心独白 / 对白 / 环境音 / 音乐卡点），再在提示词里写**谁先说、停顿多久、情绪如何变**。

- 示例：`女生第一声说"你终于承认了"，说完停顿半秒，眼眶微红但没有哭，背景只有女生和远处的汽车声`。
- 别让角色「边跑边说 / 边表演边说」——口型、动作、情绪会互相干扰；旁白镜头用背影 / 空镜 / 慢镜。
- **音频分层**：人声负责剧情、环境音负责空间、音乐负责情绪；但**别塞过多音效**——一场复仇戏只写「低频紧张配乐 + 皮鞋踩过大理石的回声」比堆「雷声 / 心跳 / 风声 / 鼓点 / 玻璃碎裂」稳得多。

### 8.2 提示词越长反而越乱 → 三层结构

模型懂自然语言逻辑，但**不等于越长越好**。用三层结构：

1. **主提示词**：只写画面核心。`女主在雨夜的酒店走廊发现男主背叛`。
2. **约束提示词**：锁角色 / 镜头 / 风格。`参考图片1中的女主脸和发型，现代都市暗调，冷蓝色雨夜，中近景，镜头缓慢推进到眼神`。
3. **禁指向**：减少常见错误。`不要改变女主服装、不要出现多余人物、不要夸张表情、不要卡通化、不要生成复杂文字`。

- **禁指向最多 3-5 个**：写太多反而污染模型的注意力 / 沾染能力。
- **每条提示词只设一个「第一优先级」**：模型算力有限，这一镜最重要的是「女主眼神」，就别同时追复杂运镜 / 暴雨闪电 / 路人 / 手机文字 / 对白字幕。先问自己「这条镜头唯一目标是什么」，给得越精准，反馈越精准。

### 8.3 人物像模特摆拍、没有心理反馈 → 写「情绪递进」并拆成可见动作

别写「伤心 / 愤怒 / 震惊」，要写**情绪递进的可见动作**：

- 难过 → `先强忍微笑，眼神短暂避开，嘴角慢慢放下，呼吸变浅，最后低头紧握手指`。
- **高级感来自克制**：虐恋 / 追妻 / 复仇戏，打动人的不是大哭而是「想哭却忍住」。提示词：`微表情优先，表演克制，眼神变化明显，嘴唇轻微颤抖，不要夸张哭喊`。
- 黑化别写「他变坏了」，写 `擦掉眼泪，表情恢复平静，眼神变冷，缓慢抬头看向镜头`；情绪走**震惊→失望→冷漠→决绝**的递进，而非突然换人格。

### 8.4 场景空间混乱 / 人物穿帮飘移 → 固定空间布局，每镜沿用

办公室 / 宴会厅 / 走廊里人物位置乱跳（上一镜门左、下一镜窗边）多是**提示词没给空间约束**。做法：先定布局——`门在画面左侧，落地窗在右后方，办公桌在中心，女主站桌前，男主站桌后`，且**每个镜头都沿用这套关系**。配合多模态：后续每条都写 `参考图片1的办公室空间布局，保持门窗、桌子和人物站立关系不变`，比单写「豪华总裁办公室」稳得多。

- **人物关系靠站位表达**：强势 / 主角靠近镜头，弱势靠后；审判者站在光里，反派站在阴影里。空间越清楚，戏越清楚。

### 8.5 多人同框容易乱、分不清谁是谁 → 分配层级

多人是 AI 视频高风险区（宴会、会议室对峙、家庭审判）。问题不是人多，而是「行动权太分散」——一镜里五个人都在争吵转身指责，模型必然面部串扰 / 手势穿帮 / 位置飘移。做法：给每人分清晰层级——

- `主动作人物`（女主，画面中央）+ `反应人物`（男主，坐右侧）+ `背景人物`（宾客，保持模糊静止）。
- 策略：**主角清晰 / 配角弱化 / 背景虚化**；三角关系用三人构图（`女主前景左侧、男主中景右侧、女配后景门口`），观众一眼看出权力关系。
- 别追求每个人都细致，**只保证主角被细描**。

### 8.6 参考素材越多画面越串味 → 一个素材一个职责

多模态≠素材越多越好。上传失败 / 串味多因「同时传女主图 + 男主图 + 场景图 + 视频 + 音乐却只写'参考这些生成'」→ 脸 / 服装 / 背景 / 动作互相污染。做法：

- **一个素材一个职责**：`图片1只参考女主脸和发型`、`图片2只参考男主服装`、`图片3只参考酒店走廊布局`、`视频1只参考女主转身动作`、`视频2只参考氛围`。
- **明确「不参考什么」**（非常关键）：`不要参考视频1中的人物外貌和场景，只参考动作和节奏`。
- 把素材当**部门分工**（角色部门 / 场景部门 / 动作部门 / 音频部门 / 镜头部门），不是灵感板。生成前先问「这个素材到底控制哪一项」，说不清就别传。

### 8.7 爽点生成出来了但「不够爽」 → 情绪落差 + 爽点三段

短剧 / 漫剧核心不是单一事件，而是**情绪落差 / 前后权力反转**。「女主打脸恶婆婆」本身不爽，**前一秒所有人看不起她、后一秒身份反转**才爽。

- 爽点三段：**压制（憋气）→ 证据（释放）→ 反应（放大快感）**。
- 写法示例：`宴会厅里所有人都在嘲笑女主，女主始终低头沉默，随后她把收购合同轻轻放在桌上，众人表情从轻蔑变成惊恐，镜头缓慢推进她平静的眼神`。
- **爽感来自周围人的反应**：别只写「女主很霸气」，要写被打脸的人怎么变脸（`反派后退 / 笑容僵住 / 酒杯发抖 / 窃窃私语停止`）。
- 镜头用 `精致压迫 + 轻微推进`；**主角动作越小、周围反应越大** → `主角动作克制，周围人反应明显，形成强烈权力反转`。别动不动爆炸 / 闪电 / 慢动作乱飞。

### 8.8 打戏 / 摔戏 / 爆破容易崩 → 省略暴力过程，强化结果反馈

动作戏是高风险内容，**别跟模型强刚极限**，也别把一整套动作塞进 15s（时间越长越易出错）。用**暗示性动作**剪出冲击力：

- 别写「男主一脚踢飞三个人」，改写 `男主眼神特写、拳头紧握 → 反派冲过来快速擦过肩膀 → 反派撞到桌子、玻璃震动`，观众自然脑补——这叫**省略暴力过程、强化结果反馈**。
- 提示词：`不要表现复杂格斗过程，只表现动作前后的瞬间冲击`。例：`反派刚冲到男主面前，下一秒画面切到反派撞击椅子，男主仍在原地、衣装整齐、表情冷静`——稳定且有爽感。
- 爆破同理：别写「汽车爆炸、人物翻滚、碎片飞溅、镜头环绕」，改写 `远处传来爆炸火光、玻璃反射橙色光，女主头发被热浪吹起，镜头保持近景`。**把危险动作放在画外，用光、声音、反应来表现**，成片更电影化。

### 8.9 竖屏不好看 / 字幕挡脸 / 主体太小 → 9:16「权力构图」，不是横屏裁切

漫剧 / 短剧大多面向**手机竖屏**，但多数提示词还是横屏思维（大场景、多人、远景、复杂背景），生出来主体太小、关键道具和手部动作落在底部、**字幕一盖就全废了**。竖屏最重要的是**脸、手、关键道具**。做法：

- **明确竖图构图**：提示词写 `9:16 竖图构图`，`人物上半身占画面主要区域`，`脸部位于画面上三分之一处`。
- **预留字幕安全区（关键）**：`底部预留字幕安全区域`——不写的话模型会把重要道具 / 手部动作塞到底部，被字幕盖住。
- **关键信息上移**：`背景保持简洁，主要在画面上半部分生成关键剧情信息`（用户在手机上的注意力集中在上半部）。
- **少超远景、多中近景**：竖屏少用超远景，多用 `中近景 / 半身 / 特写`。豪门宴会这类大场景只在开头远景**交代一次**，后续主拍人物特写 / 合同 / 眼神 / 实体动作。
- **用上下位置表达权力**：`反派站在画面上方俯视，女主坐在下方阴影里；反转后女主站起、镜头降为仰拍`。**竖屏是专为手机设计的「权力构图」，不是把横屏裁一刀**。

### 8.10 批量生成可用率低 → 先定评判标准 + 失败标签 + A/B/C 三版

工作室 / 团队一天能产十几条甚至上百条，但**真正可用率极低**——问题不是产量不够，而是**没有明确的评判标准**。做法：

- **生成前先定每条镜头的「合格条件」**：`角色一致 · 情绪明确 · 动作完整 · 构图可裁切 · 无明显穿帮`；并写清**这一镜的目的是什么**。
- **生成后按目的打分，别被「画面漂亮」蒙蔽**：短剧 / 漫剧镜头**首先要能讲故事，其次才是画面精度**。
- **建立「失败标签」复盘**：把废片标注成 `脸变了 / 糊了 / 空间错乱 / 节奏慢 / 表演假 / 文字乱码` 等——标完会发现 **80% 的问题都集中在少数几个提示词习惯上**（即你写提示词的习惯错了），针对性改即可。
- **A/B/C 三版策略 + 导演定夺**：同一镜头产三版——`A 版保守稳定`、`B 版情绪加强（加眼泪 / 呼吸）`、`C 版镜头更大胆（门缝窥视 / 推进）`；评判时**不要问「哪个更好看」**（又回到错误视角），要问「**哪个更适合这一集**」，最终决定权交给**导演**（团队里最值钱的岗位）。

### 8.11 新手不会写、不知道做什么 → 先「拆任务」，别让模型当编剧

新手最大的问题不是不会写提示词，而是**接到任务时不知道自己要做什么**。你只说「生成一个好看的短剧」，模型只能自己猜人物、猜场景、猜情绪、猜镜头、猜动作、猜节奏——猜对了夸模型厉害、猜错了怪模型不行，其实是你没把任务交代清楚。做短剧第一步**不是写提示词，是拆任务**，先想清五件事：

1. 主角是谁？　2. 主角现在在哪？　3. 主角正在经历什么？　4. 这条视频要让观众感受到什么？　5. 结尾要不要预留钩子？

- **把人物和环境立体化**，别写「女主突然很伤心」（太虚）。改写：`女主站在医院走廊，手里拿着检查单，她刚发现男主和家人一起骗了她，她没有大哭，而是强忍情绪，眼神从震惊变成冷静`——这才是 Seedance 能执行的完整命令。
- 核心一句话：**不要让模型替你当编剧**。专业提示词不在于写得多长，而在于**先把戏写明白，再让模型把它拍出来**。

### 8.12 一条视频塞太多剧情 → 一条只解决一个戏剧点（三段式）

新手第二个高频错误：在 10 秒里塞无限剧情（出事 + 出轨 + 羞辱 + 车祸 + 重生 + 复仇全堆一起）——这不是短剧，是「剧情塞车」，结果每个动作都没描绘清楚、观众看着乱、模型也不知道你要干嘛。做法：

- **一条视频只解决一个戏剧点**，系列化串联：`第1条 女主发现检查单 → 第2条 女主质问男主 → 第3条 某人出面羞辱 → 第4条 女主反击 → 第5条 孩子身份反转`。
- **每个专业短剧片段三段式**：`开头`（让观众知道发生了什么）→ `中段`（情绪 / 冲突封锁推进）→ `结尾`（留情绪爆发点或悬念钩子）。
- 把 5 集内容塞进一条 = 每个动作都崩、观众乱、模型懵；拆成「一条一戏剧点」才叫**系列化剧情**。

### 8.13 情绪写得越「过」越假 → 写情绪变化「过程」、拆成有层次的可见动作（ep17）

`女主崩溃大哭` / `男主非常生气` / `婆婆非常恶毒` 这类词不是不能用，但它们只表达**结果**、不表达「**如何表演**」。短剧 / 漫剧真正打动观众的是**情绪变化的过程**——观众想看的是「从忍住到忍不住」「从还没生气到逐渐爆发」，不是一个人直接哭 / 直接气。

- **反例（只给结果）**：`女主崩溃大哭`。
- **正解（拆成有层次的过程）**：`女主先愣住，眼神失焦，手里的检查单微微下垂；她慢慢抬头看向男主，眼眶发红，嘴唇抿紧，眼泪滑下来，但还没有大哭`。
- **为什么更好**：它有四层递进——①震惊 → ②克制 → ③情绪外露 → ④没有彻底失控，让观众悬着「他是不是马上要哭出来了」。
- 口诀：**写得越具体，角色越像在演戏；写得越抽象，角色越像在「改动作」。** 短剧里最值钱的不是「能哭能喊」，而是**让表演更像真人**。

### 8.14 人物动作没有前后逻辑 / 动作凭空发生 → 写「动作链路」三步：准备 → 触发 → 结果（ep17）

常见崩法：`女主打了男主一巴掌` → 手突然飞出去、身体没变化、男主也没反应，一眼假。根因是**动作缺少因果链路**。把每个关键动作拆成**三步动作链路**：①**准备动作** → ②**触发动作** → ③**结果动作**。

| 动作 | 三步链路写法（照抄） |
|------|----------------------|
| 打耳光 | `男主伸手想抓女主手腕（准备）→ 女主退后半步甩开他的手，盯着男主停顿一秒（触发）→ 随后抬手打了他一巴掌，男主脸被打歪、愣在原地（结果）` |
| 摔门离开 | `女主拿起桌上的离婚协议，转身走向门口（准备）→ 男主追上来喊她名字，她停顿一下没有回头（触发）→ 用力拉开门走出去，重重把门关上（结果）` |

> 原则：好提示词不在于词多高级，而在于**动作能不能让模型理解、能不能执行**。有因果的动作链路才不崩、才像真人。

### 8.15 写「电影感」没有电影感 → 把「电影感」拆成具体镜头语言（ep18）

把 `电影感` / `高级感` / `大片质感` / `氛围感拉满` 直接写进提示词**不会**让画面有电影感——模型不知道你要的「电影感」到底是低角度、前景深、逆光、慢推镜头、冷暖对比还是长焦压缩，只能猜。专业做法：**把「电影感」翻译成一段可被执行的镜头语言**。

- **反例（音乐型空词）**：`雨夜别墅门口，女主很孤独，电影感`。
- **正解（清晰的镜头表述）**：`冷蓝色雨夜，别墅门内暖黄灯光，女主站在门外，镜头慢推 + 前景深，最后特写她手里被雨水打湿的离婚协议`。
- 可调用的「电影感」零件：`低角度` / `前景深` / `逆光` / `慢推镜头` / `冷暖对比` / `长焦压缩`——按需挑 1–2 个填进画面，比写「电影感」精准得多。（与 cinematic-techniques.md 第四 / 五节光影构图、本表「要某种氛围光」互证。）

### 8.16 人物全是正面、像证件照 → 别让人物老在画面正中、用景别组合 + 遮挡叙事（ep18）

新手视频「很平」的根因：人物**一直站在画面正中间**说话，全是正面机位，像证件照 / 会议记录。短剧不是会议记录，人物可以偏一些、放在画面节点上。两个手法：

- **景别组合叙事**（不要一个景别用到底）：`远景交代地点 → 中景交代人物关系 → 近景表现人物情绪 → 特写打爆点 / 情绪点`。
  - 示例分镜：`先拍检查单 → 镜头慢慢推到他发抖的手 → 再拍他红了的眼眶 → 最后拍男主走过来的脚步`，叙事自然完整。**镜头不是叙事工具，镜头是在「替你讲故事」。**
- **用遮挡做高级感**（偷拍 / 真实视角，而非摆拍）：让前景出现 `门缝` / `玻璃` / `窗帘` / `雨伞` / `车窗反光`，从这些遮挡后面拍，会更真实、不像摆拍。
- **可套用范板（竖屏 15 秒短剧）**：`开场远景雨夜别墅门口，女主站在门外，车灯从背后打过来；镜头慢慢推进到中景，男主站在门内逆光；随后切到女主近景，她眼眶发红但没有哭；最后急推特写她手里的离婚协议，纸面被雨水打湿。冷蓝色雨光与门内暖黄光形成强烈反差`。

## 速查总表（写 Seedance 2.0 prompt 时按需取用）

- **要某种镜头** → 直接写运镜术语 + 景别变化路径（见一/二/三/四）；低角度跟拍写机位高度，希区柯克变焦写「背景被拉伸挤压」
- **要某种氛围光** → 逆光（暖调逆光 + 丁达尔）/ 暗调（三点光源 + 强明暗对比）/ 胶片（35mm + 90 年代录像带）（见五）
- **要不崩** → 加防崩神词（稳定 / 不闪烁 / 不重影 / 流畅 / 高清 / 电影质感）+ 动作写慢 + 运镜只写一个 + 优先 10 秒（见六）
- **要稳一条复杂 prompt** → 三层结构（主 + 约束 + 禁指向≤5）+ 每条一个第一优先级（见 8.2）
- **要真演技** → 情绪递进拆成可见动作 + 微表情克制（见 8.3）
- **要空间不乱 / 多人不串** → 固定布局每镜沿用 + 分层级（主动作 / 反应 / 背景）（见 8.4 / 8.5）
- **要多素材不串味** → 一素材一职责 + 明写「不参考什么」（见 8.6）
- **要爽** → 情绪落差 + 爽点三段（压制 / 证据 / 反应）+ 主角动作小周围反应大（见 8.7）
- **要打戏不崩** → 省略暴力过程、强化结果反馈、危险动作放画外（见 8.8）
- **要竖屏好看** → 9:16 竖图 + 脸在上 1/3 + 底部字幕安全区 + 关键信息上移 + 少超远景多中近景（见 8.9）
- **要量产可用率高** → 先定合格条件 + 失败标签复盘 + A/B/C 三版交导演定夺（问「哪个适合这一集」非「哪个好看」）（见 8.10）
- **新手不会写** → 先拆任务（主角是谁 / 在哪 / 经历什么 / 让观众感受到什么 / 留不留钩子），别让模型当编剧（见 8.11）
- **剧情塞车** → 一条只解决一个戏剧点 + 片段三段式（开头 / 中段 / 结尾钩子）（见 8.12）
- **情绪一眼假** → 别写结果词（崩溃大哭），写情绪变化过程，拆成有层次的可见动作（震惊→克制→外露→未失控）（见 8.13）
- **动作凭空 / 没逻辑** → 写动作链路三步：准备动作 → 触发动作 → 结果动作（见 8.14）
- **写「电影感」没电影感** → 拆成具体镜头语言（低角度 / 前景深 / 逆光 / 慢推 / 冷暖对比 / 长焦压缩）（见 8.15）
- **人物像证件照 / 画面平** → 别让人物老在正中，用景别组合（远→中→近→特）叙事 + 前景遮挡（门缝/玻璃/窗帘/雨伞/车窗反光）（见 8.16）
