# 视听语言实战技巧库（Seedance 2.0 电影感拆解）

> 来源：抖音合集「AI 视频教程」（作者：李一帆 / “AI 界人民教师一帆”，共 17 集），由视频音轨转写后蒸馏整理为可直接套用的提示词规则。原始转写见仓库外的字幕归档。
>
> **核心理念（贯穿全篇）**：别人的 AI 视频像电影、你的像“会动的 PPT”，差的不是模型、不是画质、也不是堆提示词，而是**视听语言**——用景别、运镜、光影、构图、节奏、蒙太奇去**控制观众情绪、讲清一个故事**。一句话：**不会镜头语言只是在“机械展示提示词”，会镜头语言才是在“用 AI 拍电影”。**
>
> 用法：写分镜 prompt 时，先想“这一拍要让观众**看哪里、感受到什么**”，再从下面的映射表里挑对应的镜头/光/构图/节奏词填进时间轴分镜（配合 SKILL.md 铁律 4 的对白时间轴对齐）。所有词条都是正向描述，符合铁律 5 审核规避。

## 一、Seedance 2.0 四模态能力地图（先知道工具能做什么）

Seedance 2.0（即梦 / 火山引擎；小云雀「沉浸式短片」入口）最强的是**四模态、最多 12 个参考**同时输入：

| 模态 | 参考类型 | 典型用途 |
|------|----------|----------|
| 文本 | 提示词 | 描述画面 / 动作 / 对白 / 声线 |
| 图像 | 参考图（角色三视图 / 场景图 / 分镜图 / 故事版） | 锁定角色一致性、场景、构图 |
| 视频 | 参考视频 | 镜头复刻、动作迁移、视频延长续接 |
| 音频 | 参考音频 | 锁定真实音色、配音节奏 |

由此衍生的三类“魔法级”玩法（均靠多模态组合实现）：

- **P 视频（换人换景换音）**：上传原始视频 + 想替换的人物 / 场景 / 配音参考 + 提示词 → 替换其中元素。
- **TVC 广告片**：上传 AI 生成的分镜图 + 一段口播音频 + 提示词 → 几分钟出一条完整广告片（过去需 5 人团队 3 天）。
- **电影镜头复刻**：上传想模仿的电影镜头（参考视频）+ 自制的人物 / 场景设定图 + 提示词 → 得到丝滑且原创的动作戏。

> 写多模态 prompt 时用 `@图片N`/`@视频N`/`@音频N` 引用（见多模态引用语法章节），并在【参考】区注明每个素材的用途。

## 二、景别分层：拆开信息，别塞进一个镜头

新手通病：把所有信息一次性塞进一个镜头，画面“完整”但观众**什么都抓不住**。正解是用景别分层逐步引导：

| 景别 | 作用 | 提示词触发 |
|------|------|-----------|
| 远景 | 交代环境 / 世界，让观众知道故事发生在哪 | `远景，交代…的整体环境与氛围` |
| 全景 / 中景 | 交代动作，让观众关注“谁在做什么” | `中景，跟随…走进来，交代动作` |
| 近景 | 传递情绪，让观众开始关注内心 | `近景，…停下脚步，视线被…吸引` |
| 特写 | 放大细节 / 重点，定格被打动的瞬间 | `特写，…手指轻轻贴近…的动作` |

**标准组接序列**：远景（环境）→ 中景（动作）→ 近景（情绪）→ 特写（细节）。一个“男孩走进面包店”用这套拆开，就从“一个男孩进店”变成“一个具体的人被一个具体瞬间打动”。

## 三、16 种运镜 → 情绪映射（运镜是为情绪服务，不是为了“动”）

> 口诀：推镜头让你走近人物，拉镜头让你感受世界，跟镜头让你成为角色，环绕镜头让世界围着他转。

| 运镜 | 情绪 / 作用 | 提示词触发 |
|------|-------------|-----------|
| 推镜头 | 拉近距离、放大情绪、强化信息 | `镜头缓缓推进到…的面部特写` |
| 拉镜头（作用一） | 营造孤独感、渺小感 | `镜头拉远，人物被…吞没` |
| 拉镜头（作用二） | 延迟揭示真相 / 更多信息（先误导→后恍然） | `镜头拉远，揭示出…（反转信息）` |
| 后跟 | 最强代入感 | `后跟镜头，紧跟人物背后前进` |
| 侧跟 | 交代人物与环境关系 | `侧跟镜头，平移跟随` |
| 前跟 | 突出表情与动作 | `前跟镜头，正面跟随，突出表情` |
| 360° 环绕 | 强调主角感（“全世界围着他转”） | `360 度环绕运镜，围绕主角旋转` |
| 上摇 | 强调气势、增加心理压迫感 | `上摇镜头，从下往上展现宏伟` |
| 希区柯克变焦 | 强调心理冲击 / 紧张 / 恐惧（眩晕感） | `希区柯克变焦，背景拉伸、主体不变` |
| 镜头速度·慢推 | 情绪克制、压抑、缓慢积累 | `镜头缓慢推进，情绪克制` |
| 镜头速度·快推 | 瞬间惊讶、爆发、紧张 | `镜头快速推近到面部，情绪爆发` |

> 关键认知：**决定画面故事的从来不是大模型，而是你有没有在用镜头思考。** 同一动作，慢推=克制、快推=爆发，情绪完全不同。

## 四、8 种打光 → 情绪映射（普通人用光照亮人物，导演用光塑造故事）

| 光型 | 情绪 / 作用 | 适用场景 | 提示词触发 |
|------|-------------|----------|-----------|
| 逆光 | 英雄感 / 浪漫 / 神秘（只剩轮廓） | 英雄归来、恋人拥抱、身份未知 | `逆光，照亮人物轮廓与发丝` |
| 顶光 | 压迫、权力、危险、神秘 | 黑帮老大、幕后人物 | `冷色顶光，眼窝压暗` |
| 丁达尔光 | 史诗感、神性、空间立体 | 教堂、神殿、废墟、森林 | `丁达尔光线，光束穿透空间` |
| 底光 | 诡异、不安、疯狂 / 失控 | 恐怖、人物黑化 | `底光，从下方打光打破正常光影` |
| 侧光 | 冲突感 / 内心拉扯（一半亮一半暗） | 紧张对峙、艰难抉择 | `侧光，一侧脸被照亮、另一侧隐入阴影` |
| 硬光 | 张力、攻击性、压迫感 | 拳击手、对决前 | `强烈硬光，阴影清晰、明暗对比强` |
| 柔光 | 温暖、治愈、情感温度 | 亲情陪伴、告白、回忆、病房 | `柔光，光线轻轻包裹人物` |
| 蝴蝶光 | 精致、贵气、自带主角感 | 高端酒会、女主登场 | `蝴蝶光，鼻下蝴蝶状阴影，聚光灯感` |

> 同一个场景换一种光，观众感受到的情绪就完全不同——别只写“高级感 / 大片感”，写**具体光型**。

## 五、7 种故事感构图 → 作用映射（构图为控制情绪、不是为摆好看）

| 构图 | 作用 | 提示词触发 |
|------|------|-----------|
| 前景构图 | 二维→三维空间感、沉浸感；也可模拟“偷窥”增加紧张 | `前景构图，画面前景有玻璃/树叶/台灯遮挡` |
| 中心构图 | 让人物成为绝对主角 | `中心构图，主角置于画面正中` |
| 三分法构图 | 呼吸感、自由感、空间感 | `三分法构图，主体置于三分之一处` |
| 对角线构图 | 动感与张力（速度、对抗） | `对角线构图，主体沿对角线分布` |
| 对称构图 | 秩序、权力、高级感、庄严 | `对称构图，画面左右对称` |
| 引导线构图 | 控制观众视线、空间纵深、沉浸 | `引导线构图，用线条引导视线至主体` |
| 框架构图 | 把人物“框进情绪里”，制造旁观感 / 隔离感 | `框架构图，透过门框/车窗观看人物` |

## 六、节奏：节奏 = 情绪推进速度（决定电影代入感的隐藏变量）

镜头不是在“记录动作”，而是在“控制观众情绪”。同一动作，节奏不同，感受完全不同。

**三种基础节奏**：
1. **情绪铺垫型（慢）**：少切换、给观众时间观察细节（眼神、呼吸、风、光影）。适合孤独、高级、文艺、沉浸。
2. **信息压迫型（快）**：快速切换、信息密集，让大脑来不及休息。适合动作、悬疑、追逐、紧张。
3. **先压抑再释放（商业片最爱）**：高级爽感不是一直炸，而是**控制快慢反差**。

**5 个节奏技巧**：
- **镜头前摇**：任何动作都有准备动作——开门前犹豫、说话前吸气、奔跑前重心变化。加一个前摇，人物立刻真实。
- **情绪停顿**：别全程疯狂运镜，**停顿本身就是情绪**，要敢用静止镜头。
- **情绪切镜**：切的是**情绪变化**而非动作——嘴上平静但眼神慌乱时，突然切特写放大情绪。
- **环境节奏**：让环境参与节奏——压迫感来了风变大、孤独感来了空间变空、紧张感来了灯光闪烁。
- **镜头留白 / 突然抽空**：高速追逐 / 激烈争吵 / 高潮之后突然安静（只剩耳鸣声），人类情绪对**反差**最敏感，留白反而放大前面的冲击。

> 流水账的根因：从头到尾情绪是平的（镜头一直动、人物一直说、音乐一直响）。真正高级的节奏**张弛有度**。

## 七、蒙太奇：用组接制造意义（库里肖夫效应）

同一个“男人面无表情”的镜头，后面接孩子=慈祥、接钱=贪婪、接倒下的人=危险。**同样画面 + 不同组接 = 不同意义**，这是 AI 导演最高级的造物技巧。

| 蒙太奇类型 | 作用 | 怎么拍 |
|------------|------|--------|
| 时间蒙太奇 | 用几个变化镜头压缩一段长时间 | 手机时间跳动 + 咖啡变冷 + 天色由亮变暗 + 人来了又走 |
| 情绪蒙太奇 | 不直接说情绪，用细节把情绪叠起来 | 握紧的手 + 额头的汗 + 越来越响的钟表 + 靠近的脚步声 |
| 平行蒙太奇 | 两条线同时推进，制造“能不能赶上”的悬念 | A 在人群中找人 / B 在站台上车，交叉剪辑 |
| 联想蒙太奇 | 用后一镜头改变观众对前一镜头的理解 | 平静喝茶 → 刀的冷光 → 碎酒杯 → 擦掉袖口的血 |

## 八、活人感 3 招（摆脱“一眼假”的机械感）

一眼假的根因：整段视频在**机械展示提示词**，而没有在**拍故事**。

1. **代入感**：把客观镜头改为**过肩 / 正反打 / 人物第一视角（POV）**——`改为过肩正反打镜头` / `以男生的低视角看向…`。
2. **空间感**：加**前景遮挡 + 大光圈 + 背景虚化**——`增加前景遮挡（窗 / 植物 / 小动物）`、`大光圈、背景虚化`。想不到前景就问 AI“这个场景适合加什么前景”。
3. **巧用镜头运动**：推=放大情绪、拉=揭示信息、跟=减少机械感增强代入（见运镜表）。

## 九、情绪表情公式（细腻真实、不浮夸）

把**大情绪拆成可控的小细节**，AI 就不会演过头（写“很开心”→ 露牙大笑；拆成“嘴角轻轻上扬、眼神柔和、眼睛微微弯起”→ 微表情）。

- **面部情绪 = 嘴角状态 + 眼部状态 + 面部状态**
- **完整情绪 = 表情细节 + 手部动作 + 身体姿态 + 视线方向**（四维同时存在才真实立体）

例：害羞忍笑 = `嘴角轻微上扬 + 手轻轻捂嘴 + 肩膀微微抖动 + 眼神闪烁 + 头稍微后仰`。

## 十、AI 配音 3 步（从“机器读稿”到“演员级”）

1. **用标点控制语气**：感叹号=强调 / 警告，逗号=停顿 / 思考。（“等一下，我想先确认一件事。”）
2. **别写情绪，写发声方式**：不写“紧张地说”，写发声机制——`语速放慢、声音压低、最后几个字微微上扬`。
   - 公式：**语气 = 声音大小 + 语速快慢 + 停顿位置 + 尾音变化**
3. **加场景与人物状态**：声音不是凭空出现的。
   - 公式：**场景 + 人物状态 + 发声方式 + 台词**
   - 例：`他在雨里等了很久，声音有点发颤、语速慢下来，像憋了很久才开口：“你终于来了。”`

> 与 SKILL.md 铁律 4（对白嵌入时间轴）、铁律 10（多模态 @音频 锁声线）配合使用。

## 十一、一致性硬控四步法（打破 15 秒魔咒）

AI 单段最长 ~15 秒，超过就换脸 / 飘色 / 乱位。要做到“人物统一、声音一致、位置准确、动作到位”，像真正的剧组一样建**标准资产库**：

1. **锁人物（角色资产库）**：先用图像模型（如 Nano Banana / Nano Pro / GPT-Image-2）生成角色定妆图——`画面左侧人物近景胸像、右侧全身设定图`，再展开**三视图 / 多角度**。五官、发型、服饰、神态全部固定。多个主角各生成一套备用。
2. **固定声音**：
   - *无中生有*：用 TTS（如 minimax speech）生成音频节点，设定语速 / 音色 / 音量，锁定后续复用。
   - *解声续影*：对满意的成片，剪映分离音频 → 导出 MP3 → 作为参考音频节点，沿用同一配音。
3. **限定场景（场景锚点）**：先出**空场景图**（含光影 / 色调 / 质感），生成视频时把人物 + 场景一起连接。位置易穿帮时，用**九宫格图标出人物位置**作为空间参考。
4. **动作迁移**：写不出的动作别狂抽卡——自己拍一段示范动作视频，用动作迁移模型（如可灵 3.0 动作迁移）把动作迁到设定角色上。

**续接规则**：按脚本把连续剧情拆成 15 秒片段，**用上一段的尾帧做下一段的参考**，只改动作不改场景。多个 15 秒就能无缝拼成“同一天拍摄”的连贯影片（呼应 SKILL.md 铁律 2 续接加固）。

## 十二、把感觉翻译成 AI 听得懂的语言（0 基础到大师的三层公式）

AI 不知道你说的“好看”是什么。说清三件事：**是谁 / 在什么情境 / 在干什么**。

- **主体 = 状态 + 身份**：“一个女生”太模糊 →“一个开心的高中女生”；“白领”→“自信的白领”。
- **情境 = 时间 + 地点 + 氛围**：“在咖啡馆”→“在下午阳光透进来的咖啡馆”。时间决定光线、地点决定空间、氛围决定情绪。
- **动作**：让画面从静态描述变成“正在发生的瞬间”。

合起来：`一个开心的女生，在下午阳光透进来的咖啡馆，专注地看书。`

**建认知库的方法**：看到喜欢的画面就喂给 AI 问“这张图用了什么构图 / 光线 / 景别，有什么专业名词”，记下来；创作时直接把参考图丢给 AI 拆解或“生成类似画面”，比自己干写一堆提示词更稳定高效。AI 时代真正的核心竞争力 = 能把感觉说清楚 + 认知储备 + 用认知精准指挥 AI。

## 十三、漫剧 / 短剧工业化工作流（团队 SOP 与一句话出片）

**标准化制作工序（SOP）**：故事框架与脚本 → 统一美术风格 / 角色 / 声音 → 固定场景 / 道具 / 光影色调 → 组合生成分镜 → 分段制作、剪辑成片。

**3 人团队分工（像流水线 / 包饺子）**：
- **分镜师**：把脚本拆成镜头，定每集片段数、每段情绪 / 走位 / 运镜，提前产出角色图、场景图、分镜参考图。
- **视频生成师**：按分镜图 + 标准素材，用 Seedance 2.0（全能参考）批量生成 ~15 秒片段，保证人物 / 场景 / 风格统一。
- **剪辑师**：把片段拼成完整剧情，调节奏 / 配音 / 音乐 / 字幕 / 后期。
- **协作要点**：用在线工程文件**同步推进**（分镜在拆、视频在生、剪辑在拼），把“等待 + 传文件”的时间压缩掉。比的不是单兵作战，而是流程清晰、素材标准统一。

**一句话出片的三种打法（按需选择，无优劣）**：
1. **Agent 全托管**：一句话（题材 + 风格 + 时长 + 比例 + 语言）让智能体自动完成编剧→角色→场景→分镜→生成→配乐。适合新手 / 快速出片。
2. **故事版（Storyboard）方法**：先用图像模型把全部构思整理到**一张故事版图**上，再交给 Agent 据此制作。一致性有保障、成本低、适合批量化。
3. **手动档资产库**：自建可随处调用的角色 / 场景资产库，逐镜头精确控制。适合熟练运用视听语言的专业创作者。

## 十四、AI 短剧导演思维：注意力钩子 + 角色资产库 + 分镜节点工作流

> AI 短剧「没人看」的根因不是提示词堆得不够、画面不够炫，而是**缺导演思维**。大部分人不是在做短剧，而是在「抽卡模仿」——疯狂堆清晰度、打斗戏、炫酷运镜，乍看还行，播放量却越来越低。真正稀缺的是**创意**与**对注意力的设计**：你做的是一个发布在公开平台、要参与公平竞争的短视频，而短视频最稀缺的资源**不是镜头，而是注意力**。完整链路：**剧本选题 → 角色一致性设定 → 分镜成片**，每一环都用导演思维而非抽卡心态去做。

### 14.1 五大注意力钩子（开头黄金三秒的选题武器）

强开头从来不是单纯炫技术，而是**让观众瞬间产生情绪 / 好奇**。把引导注意力的常用钩子归为五类，写开场（尤其黄金三秒）时任选其一切入：

| 钩子类型 | 机制 | 可套用开场示例 |
|----------|------|----------------|
| **异常事件** | 反常打破预期，观众不一定相信，但一定被「这怎么回事」勾着看下去 | 婚礼现场新郎突然逃婚；流浪猫突然开口说话；机器人拥有了人类情绪 |
| **情绪冲突** | 观众先对情绪产生反应、再决定要不要看剧情；别急着交代「发生了什么」，先给「人物现在是什么情绪」 | 开场即压迫感 / 崩溃感 / 愤怒感 / 羞辱感的特写与状态 |
| **信息缺口** | 故意留一块空白，观众本能想知道后面到底发生了什么；不要一上来把信息全交代 | 男主开门后突然愣住；所有人都在逃跑；女孩看完手机消息脸色瞬间变了 |
| **情绪治愈** | 用治愈感快速吸住，正向情绪同样抓注意力 | 可爱 IP 角色制作美食；小猫偷偷给熟睡主人盖被子 |
| **视觉奇观** | 超现实 / 梦幻画面制造「没见过」的惊叹 | 梦幻的鲸鱼从城市上空缓缓游过 |

> 写剧本 / 分镜前先选定钩子类型，再决定开场画面与第一句对白——这是把「核心梗」落到镜头的第一步（配合 SKILL.md 工作流第 1-3 步的核心梗 / 一句话卖点）。

### 14.2 没灵感时用 Agent 策划选题（短剧 Agent 2.0 工作流）

灵感枯竭时让短剧 Agent（即梦 / 小云雀「短剧 Agent 2.0」）辅助策划「适合短视频传播」的故事：

1. 工具首页选择 **短剧 Agent 2.0**；可上传自写剧本，无灵感则选 **AI 生剧本**。
2. 在五大钩子里挑一个写进需求，例如：「帮我生成一个**有强反转和情绪冲突**的现代 AI 短剧故事，故事开头钩子**以一个异常事件**作为开端」。
3. 展开**风格库**选视觉风格、画面比例、集数，点击立即生成 → Agent 研读剧本，从原始创意到人物设定几分钟生成。
4. 下一步自动扩展**人物关系 / 剧情反转 / 世界观**，产出**多集、带黄金三秒、剧情完整**的剧本。

### 14.3 角色资产库：让人物「被记住」而不是顶着同一张网红脸

AI 短剧通病：人物都顶着一模一样的「网红脸」，观众连谁是主角都记不住。**能被记住的人物 = 明确的视觉标签 + 情绪气质**：

- **视觉标签**：明确的五官特征（而非千篇一律的精致脸）。
- **情绪气质**：性格上的高冷感 / 破碎感 / 压迫感 / 锋利感等——这些才是让角色「有人味」的关键。

专业团队的做法是先搭**角色资产库**，把形象 / 装造 / 性格提前统一。操作（自由画布）：

1. 切到**自由画布**功能 → 点击**新建角色**。
2. **上传一张真实照片作参考**，避免生成同质化面孔。
3. 输入人物设定提示词结构，**适当加入性格特征词**；链接上传的照片后点击生成，得到完整角色设定图。
4. 右下角三个点可详编**名称 / 出现集数 / 音色**；想要音色统一就**上传准备好的声音设定**，完成后保存。
5. 左侧资产文件夹即可看到该角色；「资产库编辑」可反复修改调整。
6. 用相同方法把所有**人物 / 场景 / 道具**都生成好，方便随时调用（对应一致性硬控四步法的「角色资产库」）。

### 14.4 分镜节点：高级 AI 视频拼的是细节控制，不是生成能力

一句话抽卡已经过时；高级 AI 视频拼的是**分镜提示词的细节控制**——人站在哪里、镜头从哪个方向拍、环境是什么灯光氛围，全部提前规划，生出来的镜头才有真实感和统一性。操作（自由画布节点）：

1. 右键画布 → **添加节点** → 先建一个**视频节点**，在下方输入视频片段提示词。
2. 单段最长 **15 秒**，需**合理拆分每个 15 秒的分镜头**；用可改的提示词结构，保证多个镜头时长**加起来 ≤ 15 秒**即可较完整生成片段（对应 SKILL.md 铁律 1 时长估算）。
3. 写完提示词后**重点**：把提前备好的**角色 / 场景 / 道具**点小加号**链接到视频生成节点**，并在提示词中用 **@ 引用相应素材**（多模态引用，锁定一致性）。
4. 得到多段约 15 秒片段后剪辑成片。

> **一句话总结（ep17 核心）**：AI 短剧真正卷的不是「谁会生成」，而是「谁更像导演、更懂细节的重要性」。当别人还在疯狂抽卡，真正的 AI 创作者已经在搭建自己的 AI 剧组。

## 速查总表（写 prompt 时按需取用）

- **要有人看（开头黄金三秒）** → 选一类注意力钩子（异常事件 / 情绪冲突 / 信息缺口 / 情绪治愈 / 视觉奇观）+ 导演思维（先设计情绪与信息缺口，别抽卡炫技）
- **要讲清故事** → 景别分层（远→中→近→特）+ 蒙太奇组接
- **要情绪** → 运镜（推/拉/跟/环绕）+ 光型 + 节奏（慢铺垫 / 快压迫 / 反差释放）
- **要电影感** → 光影（逆/顶/侧/底/硬/柔/丁达尔/蝴蝶）+ 构图（前景/中心/三分/对角/对称/引导线/框架）
- **要活人感** → 过肩 POV + 前景虚化 + 镜头运动；表情拆细节 + 四维情绪公式
- **要一致性** → 角色资产库三视图 + 空场景锚点 + 尾帧续接 + 动作迁移
- **要真配音** → 标点控气 + 写发声方式 + 给场景动机
- **要量产** → SOP 分工 + 在线协作 + Agent 全托管 / 故事版 / 资产库三选一
